# DiscordDMBlocker
自動でセキュリティ措置を有効化し続けるBot
# Setup
## 動作環境
Node.js v18以降
## 起動方法
.envファイルをこのリポジトリ内の`.env.example`ファイルを参考に編集してください。
 
そしてリポジトリ直下に`.data`という名前のディレクトリを作成してください
 
あとはパッケージを`npm i`と実行してインストールしてから`node index.js`を実行すれば起動できます。
